Community

Fission is an open-source project by many contributors. We always welcome new users, explorers or contributors, via the following mediums:

Github

The fission project can be found here. Please see the Fission code of conduct that lays out what you should expect from our community. See also our guide to contributing to fission. If you encounter any issues, please report them to our Fission Issue Tracker.

Github

Slack

Join our Slack channel if you’d like to talk to us about your particular use-cases for fission, or if you need to reach us for help using fission.
